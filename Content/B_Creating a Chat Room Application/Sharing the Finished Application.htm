Sharing the Finished Application

Now that the application is finished, it is ready to be saved and shared with the intended users.

 

1. On the right menu, click theProject menu.

a. In the Title Field, type “Chat Room v.1”.

b. In the Description Field, type “Chat App created for class.”

 

2. On the top menu, click Save App.

Your application is now saved to your profile for future use.

 

3. On the top menu, click the menu icon.

 

4. Select Publish.

 

5. In the pop-up box, copy the URL and paste it into a new tab in the browser.

 

6. Click Run App in Browser.

 

7. Share the URL with your intended users.
