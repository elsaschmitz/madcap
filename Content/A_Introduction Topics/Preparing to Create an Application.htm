Preparing to create an application

Before you begin to use the Mozilla AppmakerA Mozilla program that allows users to create and share their own applications. program, familiarize yourself with the basic layout of the program and the key areas and items that will be used in the creation progress.

 

Fig. 1

Example of the Mozilla Appmaker Program.

 

Key Areas

1. Top Menu — The main navigation bar for the Mozilla Appmaker program.

2. Left Menu — The location of the brickThe basic buidling block of the Mozilla Appmaker program.s in the Appmaker program.

3.Workspace — The center space in the Appmaker program that holds the applicationA container in the Mozilla Appmaker program that holds bricks..

4. Right Menu — The location of the brick property menuA menu containing fields for modifying brick properties.and the project menuA menu containing fields for modifying the project information and name. in the Appmaker program.

 

Key Items

A. Brick — The basic building block of the Mozilla Appmaker program.

B. Tabs — Shows the number of pages open in the Mozilla Appmaker program.

C. Application — A container that holds bricks.

D. Menu Icon — A three-bar icon that displays the Mozilla Appmaker menu when clicked.

E. Join Icon — An icon that allows you to sign up for a Mozilla Appmaker account when clicked.
